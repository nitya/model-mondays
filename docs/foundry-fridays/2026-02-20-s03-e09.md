**Title:** Agent Skills for Microsoft Foundry

**Speakers:** Govind Kamtamneni

**Description:** Using coding agents for AI development requires the right activation context. Join us as we talk to Govind Kamtamneni about microsoft/skills - an open-source repository of skills, prompts, agents and MCP configurations specialized for use with Microsoft Foundry and Azure development. Learn what's available now - and what's coming next.

## Topics Discussed
- Context-Driven Development for AI
- What's in Agent Skills?
- Avoiding Context Rot
- Quickstart with `skills.sh`
- What's Next?
