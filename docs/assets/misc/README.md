# Miscellaneous Assets

This directory contains season banners, logos, and other shared assets.

## File Types

### Season Banners
- **Naming**: `season-{number}-banner.{ext}`
- **Example**: `season-3-banner.png`
- **Size**: 1920x1080px (16:9)
- **Format**: PNG
- **Usage**: Season announcements, overview pages

### Logos and Branding
- **Naming**: `model-mondays-logo.{ext}`
- **Formats**: SVG (preferred), PNG
- **Variants**: Include light and dark versions if needed
- **Usage**: Headers, promotional materials

### Shared Graphics
- **Naming**: Descriptive names (e.g., `cta-register.png`, `divider.svg`)
- **Usage**: Reusable UI elements across seasons

## Image Specifications

- **Season Banners**: 1920x1080px, PNG, max 2MB
- **Logos**: Scalable SVG preferred, PNG with transparent background
- **Icons**: 512x512px, PNG with transparency
- **Max file size**: 2MB for banners, 500KB for logos/icons

## Adding New Shared Assets

1. Use descriptive, lowercase, hyphenated names
2. Include version number if updating existing assets
3. Update references in `seasons.json` or other data files as needed
4. Keep original source files (AI, Sketch, Figma) in a separate design repository
