Foundry IQ is a unified knowledge layer powered by Azure AI Search, allowing agents to query and retrieve data for context engineering. Discover how it transforms agent capabilities with intelligent knowledge access.

**Key Features:**
- Unified knowledge layer for agents
- Azure AI Search integration
- Advanced retrieval capabilities
- Context engineering tools
- Enterprise knowledge management
